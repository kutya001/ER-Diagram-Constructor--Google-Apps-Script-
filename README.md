# ER Diagram Constructor

Визуальный конструктор ER-диаграмм, работающий как веб-приложение Google Apps Script. Все данные хранятся в Google Таблицах — без внешних зависимостей и установки.

## Возможности

- **Схемы БД** — создание, копирование, удаление схем
- **Интерактивный канвас** — drag & drop таблиц, zoom/pan, мини-карта, **интеллектуальная подсветка** связанных сущностей при выборе
- **Авторасположение** — 5 алгоритмов (умное по FK, сетка, радиальное, по категориям, компактное)
- **Таблицы и колонки** — полный CRUD с типизацией (PK, FK, INT, STR, DATE, UUID, JSON и др.), поддержка **поля Примечание (note)**
- **Визуальные связи** — FK-связи рисуются на канвасе, создаются перетаскиванием
- **Шаблоны** — переиспользуемые шаблоны таблиц с предустановленными колонками
- **Справочники** — управление категориями (с выбором цвета), назначениями, типами колонок
- **JSON** — экспорт/импорт схем, генерация промптов для создания схем через AI
- **Экспорт в Google Таблицу** — создание отдельной таблицы из схемы с опциональными примерами данных
- **Sheet Builder** — управление листами Google Таблицы, привязанными к схеме
- **3 темы** — тёмная, светлая, тёплая
- **Синхронизация** — real-time индикация статуса, пакетная обработка столбцов, **оптимистичный UI** для мгновенного отклика

## Технологии

| Слой | Технология |
|------|------------|
| Среда выполнения | Google Apps Script (V8) |
| Бэкенд | `.gs` файлы (серверный JavaScript) |
| Фронтенд | Vanilla JS + Canvas API, inline в `.html` |
| Хранилище | Google Sheets (8 листов) |
| Стилизация | CSS Custom Properties (темизация) |
| Деплой | GAS Web App (`doGet`) |
| Локальная разработка | [clasp](https://github.com/google/clasp) |

## Модель данных

Данные хранятся в 8 листах Google Таблицы:

| Лист | Описание |
|------|----------|
| Категории таблиц | Категории для группировки таблиц |
| Назначение таблиц | Назначения / роли таблиц |
| Типы колонок | Типы данных (PK, FK, INT, BIGINT, FLOAT, BOOL, STR, TEXT, DATE, TIME, DATETIME, UUID, JSON) |
| Схемы Баз данных | Схемы с позициями камеры |
| Таблицы | Таблицы внутри схем с позициями на канвасе, описанием и примечаниями (note) |
| Столбцы Таблиц | Колонки таблиц (тип, PK/FK, ссылки) |
| Шаблоны таблиц | Шаблоны для быстрого создания таблиц |
| Столбцы шаблонов | Колонки шаблонов |

## Установка и деплой

### Требования

- Аккаунт Google
- [Node.js](https://nodejs.org/) + [clasp](https://github.com/google/clasp)

### Шаги

1. Установите clasp:
   ```bash
   npm install -g @google/clasp
   clasp login
   ```

2. Клонируйте репозиторий и перейдите в папку проекта.

3. Создайте новый GAS-проект, привязанный к Google Таблице:
   ```bash
   clasp create --type sheets --title "ER Diagram Constructor"
   ```

4. Отправьте код:
   ```bash
   clasp push
   ```

5. Откройте редактор скриптов:
   ```bash
   clasp open
   ```

6. В редакторе: **Развернуть** → **Новое развёртывание** → **Веб-приложение**
   - Выполнять как: **Я**
   - Доступ: по вашему выбору

7. Откройте URL развёртывания — приложение автоматически создаст все листы и заполнит справочники.
